# 🍺 Собутыльник — Telegram-бот

Сервис для поиска компании, чтобы скоротать вечер вместе. Публикуй объявления о том, что у тебя есть выпить, находи людей рядом по геолокации, встречайся и оставляй честные отзывы.

---

## Возможности

- **Объявления** — создай объявление с описанием выпивки, закуски и геолокацией
- **Поиск** — найди объявления в радиусе 5 км от тебя
- **Отклики** — откликнись на чужое объявление с личным сообщением
- **Встречи** — автор принимает/отклоняет заявки, создаётся встреча
- **Отзывы** — после встречи обе стороны оставляют отзыв (1–5 звёзд + текст)
- **Рейтинг** — автоматически пересчитывается на основе всех отзывов
- **Честность** — отзывы нельзя редактировать или удалять

---

## Быстрый старт

### 1. Получи токен бота

1. Открой Telegram, найди [@BotFather](https://t.me/BotFather)
2. Отправь `/newbot` и следуй инструкциям
3. Скопируй полученный токен

### 2. Настрой конфигурацию

```bash
cd sobutylnik
cp .env.example .env
# Открой .env и вставь свой токен:
# BOT_TOKEN=1234567890:AABBCCDDEEFFaabbccddeeff
```

### 3. Установи зависимости

```bash
pip install -r requirements.txt
```

### 4. Запусти бота

```bash
python3 bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Регистрация / главное меню |
| `/profile` | Мой профиль и рейтинг |
| `/post` | Создать объявление |
| `/mylistings` | Мои объявления |
| `/find` | Найти собутыльника рядом |
| `/meetings` | Мои встречи |
| `/reviews` | Мои отзывы |
| `/review_<id>` | Оставить отзыв о встрече |
| `/help` | Помощь |

---

## Сценарий использования

```
Пользователь А (хозяин):
  1. /post → заполняет форму → публикует объявление

Пользователь Б (гость):
  2. /find → отправляет геолокацию → видит объявление А
  3. Нажимает «Хочу присоединиться» → пишет сообщение

Пользователь А:
  4. Получает уведомление с профилем Б
  5. Нажимает «Принять» → создаётся встреча, объявление закрывается

Оба:
  6. Встречаются в реальной жизни 🍻
  7. /meetings → «Завершить встречу»
  8. /review_<id> → оставляют отзывы друг о друге
```

## Правила сервиса

- Только для совершеннолетних (18+)
- Отзывы нельзя редактировать или удалять
- Объявления автоматически истекают через **6 часов**
- Уважай других участников

## Деплой на сервер (опционально)

```bash
# Systemd сервис
sudo nano /etc/systemd/system/sobutylnik.service
```

```ini
[Unit]
Description=Sobutylnik Telegram Bot
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/home/ubuntu/sobutylnik
ExecStart=/usr/bin/python3 bot.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable sobutylnik
sudo systemctl start sobutylnik
sudo systemctl status sobutylnik
```
